max'fn   |функция возвращающая максимум двух целочисленных чисел
  params: a, b,
  cmp: a' ?? b'         |сравнить a' и b'
    >=:  a' -> return   |если больше или равно то из функции вернётся a'
  b' -> return          |или b' в остальных случаях.


|Ключевое слово cmp:
|производит сравнение чисел разделённых знаком "??" в первом подблоке.
|И переводит упровление в код подблока с соответствующим знаком сравнения.
|В конце каждого такого подблока есть неявный выход обратно в основной код.
