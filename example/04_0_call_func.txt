|Для обьявления функции используется директива - 'fn
|Как было сказано выше, программа - рассматривается как функция. 
|Для этого все строки программы рассматриваются как подблоки общего блока c директивой 'fn
|(и они могут содержать в нутри себя свои блоки с 'fn)

awoo'fn             # функция awoo
  0 -> return       # здесь return это не выход из программы, а выход из awoo

-> awoo -> return   # вызов функции awoo и выход из программы.
